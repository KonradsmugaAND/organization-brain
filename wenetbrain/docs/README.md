# WenetBrain — Dokumentacja deweloperska

> Źródło prawdy dla każdego, kto rozwija WenetBrain — człowieka lub agenta AI.  
> Przed każdą zmianą w agentach, promptach lub tools przeczytaj odpowiedni rozdział i wypełnij checklistę.

| Dokument | Zawartość |

| [`agents/index.md`](./agents/index.md) | Lista wszystkich agentów runtime |
| [`agents/*.md`](./agents/) | Szczegółowy opis + checklista modyfikacji per agent |
| [`acl.md`](./acl.md) | Macierz ACL (źródło prawdy dla uprawnień) |
| [`workflows.md`](./workflows.md) | Development workflow i przydatne komendy (backend + frontend) |
